## Decisão

Quando o usuário registra uma **conquista** (modo `vitoria` no schema
`diario_emocional` ou `modo: positivo` no schema `evento`), **deve
ser obrigatório anexar pelo menos uma mídia**. Para dias negativos
(`trigger`/`negativo`), mídia é **opcional** — pedir mídia em
momento de crise adiciona fricção indevida.

A ideia: conquistas são marcos que merecem peso emocional duradouro.
Forçar a captura de mídia evita registros vazios e dá material rico
para o calendário/timeline futura.

## Tipos de mídia aceitos

1. **Spotify** — link da música (e idealmente busca dentro do app).
2. **YouTube** — link colado, extrai videoId, mostra thumbnail.
3. **Foto** — galeria ou câmera (já temos `expo-image-picker` da M03.2).
4. **Áudio** — gravação on-device (`expo-av` já instalado em M01.1).

## Implicação no schema

```yaml
# eventos/YYYY-MM-DD-slug.md (positivo)
midia:                          # array obrigatório se modo=positivo
  - tipo: spotify
    track_id: 5xQ...
    titulo: Music name
    artist: Artist name
  - tipo: youtube
    video_id: dQw4...
    titulo: Video title
  - tipo: foto
    path: ./assets/2026-04-29-conquista.jpg
  - tipo: audio
    path: ./assets/2026-04-29-1430.m4a
    duracao_seg: 23
```

`EventoSchema` ganha `midia: z.array(MidiaSchema).min(1)` quando
`modo === 'positivo'`. Refinement (`z.refine`). Mesmo no
`DiarioEmocionalSchema` para `modo === 'vitoria'`.

## UI implicada

- Componente `<MidiaPicker>` novo com 4 abas (Spotify / YouTube /
  Foto / Áudio).
- **Spotify**: exige Web API com PKCE flow + token em SecureStore.
  Avaliar custo vs benefício; opção mais simples = aceitar só link
  colado e fazer scrape de oEmbed para metadata (sem auth).
- **YouTube**: aceita link colado, extrai videoId via regex,
  thumbnail em `https://img.youtube.com/vi/{id}/hqdefault.jpg`.
- **Foto**: `expo-image-picker` (já instalado).
- **Áudio**: `expo-av` (já instalado) — gravar até 60s, salvar em
  `~/<Vault>/assets/<timestamp>.m4a`.

thumbnail YT, album art Spotify, ou waveform de áudio). Tela nova,
provavelmente em M11.x ou separada.
