# Sprint M39.1 — Migrar 9 writers para escreverMidiaComCompanion

```
DEPENDE:    M39 (helpers canônicos prontos)
BLOQUEIA:   nada
ESTIMATIVA: 2-3h
PRIORIDADE: média (consolidação técnica; backward-compat já está OK)
```

## 1. Achado / motivação

M39 entregou:
- `MidiaCompanionSchema` zod canônico em `src/lib/schemas/midia-companion.ts`.
- `escreverMidiaComCompanion(vaultRoot, binarioUri, meta)` em
  `src/lib/vault/midiaCompanion.ts`.
- `lerCompanion()` + `migrarAssetsLegacyParaMedia()`.

Mas **9 writers existentes ainda escrevem binário + companion
manualmente** via `stringifyCompanionMidia` legado:
1. `src/lib/midia/capturarFoto.ts`
2. `src/lib/midia/capturarMusica.ts`
3. `src/lib/midia/capturarVideo.ts`
4. `src/lib/midia/salvarFrase.ts`
5. `src/lib/diario/recordAudio.ts`
6. `src/lib/eventos/saveEvento.ts:copiarFotos`
7. `app/medidas/novo.tsx`
8. `src/lib/scanner/saveNota.ts`
9. `src/lib/midia/adicionarFotoManual.ts`

Decisão deliberada de M39: NÃO migrar inline para evitar quebrar
ordem determinística de testes M34 e backward-compat dos 4 fixes
A3.x.1-4 recém-fechados.

## 2. Objetivo

Migrar incrementalmente os 9 writers para usarem
`escreverMidiaComCompanion` em vez de gerenciar binário + companion
manualmente. Reduz duplicação, garante validação zod no caminho de
escrita, prepara terreno para deprecation eventual de
`stringifyCompanionMidia`.

## 3. Estratégia (1 writer por commit)

Cada writer migra independentemente. Ordem sugerida (do mais simples
ao mais complexo):
1. `salvarFrase.ts` — só `.md`, sem binário (caso especial; pode
   manter stringifier).
2. `capturarFoto.ts` / `capturarMusica.ts` / `capturarVideo.ts` —
   wrappers de picker, similar uns aos outros.
3. `adicionarFotoManual.ts` — wrapper que delega.
4. `recordAudio.ts` — usa Audio API.
5. `copiarFotos` em `saveEvento.ts` — loop de N fotos.
6. `medidas/novo.tsx` — 3 fotos com `medida_ref`.
7. `saveNota.ts` — PDF + wikilink em outro `.md`.

## 4. Verificação

- Cada writer tem teste Jest existente; manter passing.
- Cada migração via PR/commit isolado para facilitar rollback.
- Hook PT-BR + Gauntlet leak + smoke verde após cada step.
- Bundle Hermes não pode subir mais que 50 KB no total
  (estamos em 8.84 / limite 8.85 — atenção crítica).

## 5. Decisões pendentes

- **`salvarFrase` muda?** Frase só tem `.md`, sem binário. Caso
  especial. Decisão na execução: manter ou criar variante
  `escreverApenasCompanion(vaultRoot, meta)`.
- **`saveNota` PDF + .md semântico em `inbox/`** — duplo write
  com wikilink. Adaptar `escreverMidiaComCompanion` ou manter
  manual.
