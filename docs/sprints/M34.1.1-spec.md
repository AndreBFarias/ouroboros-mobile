## 1. Achado / motivação

M34.1 elevou o z-index do `<BottomSheet>` para 100 e aplicou
`zIndex: 50` no wrapper do MenuCapturaVerde. Validação Gauntlet
2026-05-04 confirmou que o aumento teve efeito DENTRO do stacking
context interno, mas:

- `<FABMenu>` é renderizado em `_layout.tsx` raiz como **irmão** do
  `<Stack>`.
- O sheet inline do MenuCapturaVerde está dentro do Stack
  (descendente da rota `/memoria`).
- Em CSS web, comparação de stacking contexts é feita no nível do
  ANCESTOR comum. Aqui o `<Stack>` (z-auto) compete com `<FABMenu>`
  (z=10) — FAB vence, e o sheet z=100 só vale dentro do Stack.

Resultado: ao rolar o ScrollView interno do `<SheetFrase>` até o
botão Cancelar, FAB roxo continua visualmente sobreposto.

## 2. Caminho

Implementar o **caminho B** descartado na spec original M34.1 por
ser "mais invasivo" (que se provou ser o único caminho viável):

```ts
// src/lib/stores/useNavegacao.ts (existente):
interface NavegacaoState {
  ...
  menuLateralAberto: boolean;
  sheetCapturaAberto: boolean;  // novo
  setSheetCapturaAberto: (v: boolean) => void;  // novo
}
```

```tsx
// MenuCapturaVerde.tsx:
const setSheet = useNavegacao((s) => s.setSheetCapturaAberto);
const abrirMenu = useCallback(() => {
  setSheet(true);
  haptics.fab();
  sheetRef.current?.expand();
}, [setSheet]);

const fecharMenu = useCallback(() => {
  setSheet(false);
  sheetRef.current?.close();
}, [setSheet]);

// E o `onChange` do BottomSheet também sincroniza setSheet(false)
// quando o usuário fecha por gesto (pan-down).
```

```tsx
// FABMenu.tsx:
const sheetCaptura = useNavegacao((s) => s.sheetCapturaAberto);
if (rotaEsconderFAB(rota) || sheetCaptura) return null;
```

## 4. Verificação

- `npm test --silent` passa com +2 testes.
- TS strict 0.
- Gauntlet: `/memoria` → tap FAB verde → FAB roxo ausente do DOM
  → tap "capturar frase" → sheet montado, sem FAB roxo cobrindo
  Cancelar → tap Cancelar funciona → FAB roxo volta.

## 5. Decisões tomadas

- **Flag em `useNavegacao` (já existente)**: mesma store que governa
  MenuLateral aberto. Coerente.
- **Sincronização via `onChange` do gorhom**: gesto pan-down também
  fecha; flag precisa cair junto.
- **Não esconder MenuLateral**: MenuLateral é navegação, sheet de
  captura é registro. Coexistir não faz sentido (dois sheets), mas
  esconder MenuLateral ao abrir sheet seria sobre-escopo.
